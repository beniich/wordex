CUSTOM_SQL

Executes a direct structured query language command against connected datasets. Ideal for complex relational operations that exceed standard cell-based logic capabilities.

code Syntax

CUSTOM_SQL ( "query_string" , [range_parameters] )

Arguments

query_string

The SQL command to execute. Supports standard ANSI SQL. Wrap table names in double quotes for sheet references.

range_parameters Optional

An optional array of cell ranges to inject as variables into the query using the ? placeholder.
