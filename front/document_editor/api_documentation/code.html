Endpoints API

Seamlessly integrate the Aether Suite into your existing workflows. Our RESTful API follows standard architectural patterns for maximum celestial efficiency.

Auth

POST /v2/auth/login
Requires: None

Authenticate User

Generate a secure session token to access protected workspace resources. Tokens are valid for 24 hours from issue date.

Request Body

email string User's registered email address.
password string The secure account password.

Workspaces

GET /v2/workspaces
Requires: Bearer Auth

List All Workspaces

Retrieve a paginated list of all workspaces accessible to the authenticated user account.

info

Use the ?limit= and ?offset= parameters to control pagination flow. Default limit is 25.

Documents

PATCH /v2/documents/{id}
Requires: Bearer Auth

Update Document Metadata

Perform partial updates to a document's core metadata, tags, or visibility settings.
